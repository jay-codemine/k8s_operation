## Deployments.vue 修改计划

### 需要修改的模态框
1. showCreateModal (L512)
2. showEditModal (L1386) 
3. showYamlModal (L2287)

### 修改步骤
1. 在 script 开头添加 import
   ```js
   import { useResizableModal } from '@/composables/useResizableModal'
   ```

2. 在状态定义区域添加三个 composable 调用
   ```js
   // 创建模态框可拖拽
   const {
     modalRef: createModalRef,
     modalStyle: createModalStyle,
     startResize: createStartResize
   } = useResizableModal({ initialWidth: '900px' })
   
   // 编辑模态框可拖拽
   const {
     modalRef: editModalRef,
     modalStyle: editModalStyle,
     startResize: editStartResize
   } = useResizableModal({ initialWidth: '900px' })
   
   // YAML模态框可拖拽
   const {
     modalRef: yamlModalRef,
     modalStyle: yamlModalStyle,
     startResize: yamlStartResize
   } = useResizableModal({ initialWidth: '1000px' })
   ```

3. 修改创建模态框 HTML (L513)
   将 `<div class="modal-content modal-create-deployment">` 
   改为 `<div ref="createModalRef" class="modal-content modal-create-deployment resizable-modal" :style="createModalStyle">`
   
   在 modal-header 之前添加 8个拖拽手柄

4. 在 style 标签添加
   ```vue
   @import '@/assets/styles/resizable-modal.css';
   ```

## 通用修改模板

对于任何视图，遵循以下模板：

### JavaScript 部分
```javascript
// 1. 导入
import { useResizableModal } from '@/composables/useResizableModal'

// 2. 在 setup 中调用（为每个模态框）
const {
  modalRef: xxxModalRef,
  modalStyle: xxxModalStyle,
  startResize: xxxStartResize
} = useResizableModal({
  initialWidth: '800px',
  initialHeight: 'auto'
})
```

### HTML 部分
```vue
<!-- 原来 -->
<div v-if="showXxxModal" class="modal-overlay">
  <div class="modal-content">
    ...
  </div>
</div>

<!-- 改为 -->
<div v-if="showXxxModal" class="modal-overlay">
  <div 
    ref="xxxModalRef"
    class="modal-content resizable-modal"
    :style="xxxModalStyle"
  >
    <!-- 8个拖拽手柄 -->
    <div class="resize-handle resize-handle-top" @mousedown="xxxStartResize($event, 'top')"></div>
    <div class="resize-handle resize-handle-bottom" @mousedown="xxxStartResize($event, 'bottom')"></div>
    <div class="resize-handle resize-handle-left" @mousedown="xxxStartResize($event, 'left')"></div>
    <div class="resize-handle resize-handle-right" @mousedown="xxxStartResize($event, 'right')"></div>
    <div class="resize-handle resize-handle-top-left" @mousedown="xxxStartResize($event, 'top-left')"></div>
    <div class="resize-handle resize-handle-top-right" @mousedown="xxxStartResize($event, 'top-right')"></div>
    <div class="resize-handle resize-handle-bottom-left" @mousedown="xxxStartResize($event, 'bottom-left')"></div>
    <div class="resize-handle resize-handle-bottom-right" @mousedown="xxxStartResize($event, 'bottom-right')"></div>
    
    <!-- 原有内容 -->
    ...
  </div>
</div>
```

### CSS 部分
```vue
<style scoped>
@import '@/assets/styles/resizable-modal.css';

/* 原有样式 */
</style>
```

## 自动化脚本（可选）

如果需要，可以编写 Node.js 脚本自动批量修改：

```javascript
const fs = require('fs')
const path = require('path')

const viewsToModify = [
  'Deployments.vue',
  'Statefulsets.vue',
  // ...
]

viewsToModify.forEach(file => {
  const filePath = path.join(__dirname, 'k8s-web/src/views/workloads', file)
  let content = fs.readFileSync(filePath, 'utf-8')
  
  // 添加 import
  if (!content.includes('useResizableModal')) {
    content = content.replace(
      /import { ref/,
      "import { useResizableModal } from '@/composables/useResizableModal'\nimport { ref"
    )
  }
  
  // ... 其他修改逻辑
  
  fs.writeFileSync(filePath, content, 'utf-8')
  console.log(`✅ ${file} 已更新`)
})
```

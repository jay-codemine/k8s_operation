# UI 优化版本回滚指南

## 版本信息
- **v1.0 (稳定版)**: 基础分页功能，Emoji 图标，简洁蓝色系
- **v2.0 (专业版)**: 渐变色彩，微动效，专业图标，大厂风格

## 快速回滚到 v1.0

### 方法 1：移除增强样式文件（最简单）⭐

**步骤：**
1. 打开 `k8s-web/src/main.js`
2. 删除或注释这一行：
   ```javascript
   // import './styles/enhancement.css'
   ```
3. 保存文件，刷新浏览器

**效果：** 立即回到 v1.0 基础样式

---

### 方法 2：使用 Git 回滚（完整回滚）

```bash
# 查看提交历史
git log --oneline

# 回滚到 v1.0 标记（如果已创建）
git reset --hard v1.0-stable-ui

# 或回滚到指定提交
git reset --hard <commit-hash>
```

---

### 方法 3：删除增强样式文件

```bash
# 删除增强样式文件
rm k8s-web/src/styles/enhancement.css

# 同时需要从 main.js 移除导入行
```

## 常见问题

### Q: 回滚后样式还是新的？
A: 清除浏览器缓存（Ctrl+Shift+R 或 Cmd+Shift+R）

### Q: 部分样式没有生效？
A: 检查控制台是否有 CSS 错误，确保文件路径正确

### Q: 想保留部分 v2.0 样式？
A: 打开 `enhancement.css`，删除不需要的样式块即可

## 技术支持

如有问题，请检查：
1. `main.js` 中的导入语句
2. `enhancement.css` 文件是否存在
3. 浏览器控制台是否有错误
4. 是否清除了浏览器缓存
